CSS styles html documents by selecting an html tag with selectors and giving the selected html tags rule sets to give the tag properties such as color and position on the page. This is done by making an external CSS file. After giving it a name, link the name of the CSS file into the head of the html document with the html tag "link" and the attribute "href". After that, type an equal sign and a quotation marks with the name of the CSS file with another quotation mark after it. At this step, add the attribute "rel" with an equal sign and type stylesheet with quotation marks. Lasty, add the attribute type with a equal sign after it with the text text/css in quotation marks ending with a backslash and a closing angle bracket. When this is completed, you selected tags that you wish to manipulate. This is done by adding a selector which is the name of html tag then adding a property to choose what about the html tag you want to change with a value which dictates how the property is changed.
