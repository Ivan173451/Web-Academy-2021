What is a Favicon?

A favicon is an image in a tab. The favicon is also known as theshortcut icon, website icon, tab icon, URL icon, or bookmark icon. A favicon is important to a website as it is one of the first images that identifies a website. A favicon is very helpful on making websites recognizable and makes it easier to know the diffences between websites. The image that makes the website easier to identify also can make people that are new to the website more likely to look at the website as it may give a good first impression and be interesting to a person seeing the favicon. The favicon is basically a book cover for a website.

How are Favicons created?

Favicons are very easy to make as long as there is a art piece or some form of design that can be used. Then, convert the image to a file type that can be displayed thorugh code. Then you add the image to the website through code that displays the image as a favicon on the right of the tab.

How did I create the Favicon for this site?

The favicon on this website was made using html. The code was placed after the line < head > with the code being < link rel="shortcut icon" href="logo.png" type="image/x-icon" >. The secton of the code that shows href="logo.png" is the image. It gets the image by getting directed to the folder that has the image file then it will display the image file. After that, the code < /head > is typed after the line < title >
