What is inline CSS?

Inline CSS is CSS that is used within an html element by using the attribute "style" and giving the attribute a value. The attribute should be placed after an html tag. This CSS can be used within an html file without the use of another file that has to be connected to the html file. The inline CSS is capable of modifying html elements such as color.

What is internal CSS?

Internal CSS is CSS that is used by inputing html elements within a style tag that is in the head of the html file. This form of CSS doesn't require another file to modify the html elements due to the fact that interal CSS is within the html file. This form of CSS can modify the postioning of an html element and it's color.

What is external CSS?

External CSS is CSS that requires a stylesheet to modify the elements within an html file. This is done by using a selector to choose what html elements or element you want to modify then giving the selector a rule set to tell the html file how to modify the html element. However, modifying the elements, you must link the external stylesheet with the html file in order for the selectors and rule sets in the CSS to work.
