To make a hyperlink, your need to type down < a h r e f = " then add the website URl. After that you type a " and a closing angle bracket. After that, you can type down text that will bring up the website when it is typed or you can add a picture that will open the website when clicked on. To add image , you need to type < i m g s r c = " after the closing angle bracket of the website typed down. Then type down the picture file name after the " adding the file extension and typing a ". Then type down w i d t h = " 2 5 0 " > to make the size of the picture. Finally after that, add < / a > to end the line.
